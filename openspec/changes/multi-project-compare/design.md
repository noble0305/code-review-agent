# Design: 多项目对比仪表盘

## 方案
在 History Tab 中，统计卡片和趋势图之间插入一个「项目对比」折叠面板。

### 组件
1. **项目选择器**：从 /api/history 提取去重 project_path，渲染为 checkbox 列表
2. **横向对比柱状图**：每个选中项目取最新评分，横向柱状图
3. **趋势叠加折线图**：多项目分数按时间叠加显示
4. **对比详情表格**：项目名、最新分、平均分、分析次数、趋势（↑↓→）

### 数据流
```
/api/history → 提取唯一项目 → 用户勾选 → 过滤数据 → 渲染图表
```

## 验证标准
- [ ] 项目列表正确去重
- [ ] 勾选项目后图表实时更新
- [ ] 无数据时不报错
- [ ] 现有趋势图不受影响
